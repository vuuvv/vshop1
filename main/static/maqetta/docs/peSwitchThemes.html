Switching Themes

Maqetta Preview 1 provides the following themes:

  • claro - the default theme, equivalent to the Dojo Claro theme
  • sketch - for wireframing, simulates a sketched design

To switch themes:

  1. Click on the Switch theme icon in the Maqetta toolbar.
  2. In the Select Theme dialog, select one of the three themes from the drop-down.
