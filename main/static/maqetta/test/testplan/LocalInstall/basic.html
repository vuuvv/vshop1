Local install basic tests

Pull down the most recent daVinci build

Steps

  1. Open a browser window and go to the following Web page:
  2. Find the relevant ZIP file for the current version (e.g., daVinci_04.zip)
  3. Download the ZIP file from the browser by using the Save Link As... (or similar) command on the context menu (right mouse button on Windows), placing the ZIP file somewhere on your local disk.
  4. After downloading, unzip the contents into a directory somewhere on your system.

Expected results

  1. The directory should contain the following files:
    • configuration/
    • .eclipseproduct/
    • davinci.local.mac.command
    • davinci.local.win.bat
    • davinci.server.mac.command
    • davinci.server.win.bat
    • eclipse.exe
    • plugins/

Launch the daVinci server (need to run tests on both Windows and Mac)

Steps

  1. With the Explorer (Windows) or Finder (Mac), navigate to the directory into which you unzip-d the daVinci files
  2. Launch the daVinci server process:
    • On Windows, double-click on the file davinci.local.win.bat (Depending on your settings, the Windows explorer might not show the ".bat" extension)
    • On Mac, double-click on the file davinci.local.mac.command (Depending on your settings, the Mac finder might not show the ".command" extension)

Expected results

  1. You can tell that the server has launched if you see the following message in the shell window: 
    !!!!!!!!!!!!!!
NOTE: CLOSING THIS WINDOW WILL 
      STOP THE DAVINCI SERVER PROCESS
!!!!!!!!!!!!!

Using directory: .....

osgi>
  2. The daVinci server will continue to run until you close the shell window that comes up after you double-click on the server launch file. Therefore, if you close the shell window, the daVinci server process will stop running, at which point you can't run http://localhost:8080/davinci anymore.
