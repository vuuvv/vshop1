Theme editor basic tests

Primitive widget theming

Steps

  1. You'll want to setup a virgin user environment by creating a whole new user login or resetting the existing user login (e.g., delete users/ folder, restart server). Clear the browser cache. Launch the daVinci server. Login if necessary.
  2. Open the Dojo/Claro theme editor from the daVinci menu bar using "Open > Open Claro Theme Editor".
  3. From the states tab select the Normal state. All widgets should be enabled
  4. Click to select the TextBox widget. In the Styling palette, change some properties to restyle TextBox widgets. For example, change some combination of font-family, font-size, font-color, boldness, padding, border color/thickness and backgrounds. You should see the effect on the canvas as you change properties.
  5. Click to select the Button widget. In the Styling palette, change some properties to restyle button widgets. For example, change some combination of font-family, font-size, font-color, boldness, padding, border color/thickness and backgrounds. You should see the effect on the canvas as you change properties.
  6. Change the state of the widget, for example from the States tab select the Hover state. If a state is selected that is not supported by the widget the widget will be disabled, visually grayed out.
  7. Repeat steps 4 and 5 for every state that that the widget supports (is not disabled).
  8. Click on the Save icon on the main toolbar to save the new file. The asterisks that indicates that the file is dirty should go away.
  9. Click on the X close button for the editor tab for this file
  10. Re-open the theme editor.
  11. Press the Add file icon on the main toolbar to create a new HTML file.
  12. Add a Button and TextBox widget to the canvas.

Expected results

  1. As you change states widgets should be disabled that do not support that state.
  2. As you change properties in the theme editor, the canvas should update so that the widgets reflect the styling changes you made.
  3. After save and reopen, the saved theme should open and its contents should be the same as how it was when you clicked on the Save button.
  4. After you have created your new HTML file with the widgets, you should see that the widgets have the styling changes you made in the theme editor.

Global theming

Steps

  1. You'll want to setup a virgin user environment by creating a whole new user login or resetting the existing user login (e.g., delete users/ folder, restart server). Clear the browser cache. Launch the daVinci server. Login if necessary.
  2. Open the Dojo/Claro theme editor from the Open menu.
  3. From the states tab select the Normal state. All widgets should be enabled
  4. Click to select the generic text widget. In the Styling palette, change some properties. For example, change some combination of font-family, font-size, font-color, boldness, padding, border color/thickness and backgrounds.
    You should see the effect on the canvas as you change properties. Some of the property changes such as font-family and font-size should affect multiple widgets.
  5. Select the Global Backgrounds and Borders. widget. This is the tab container in the global styling area. The Normal state must be selected. Change background-color to light green.
    You should see the effect on multiple widgets.
  6. Over time, we will be adding other widgets to the Global styling area. Click to select some of these other global widgets.In the Styling palette, change some properties. You should see the effect on the canvas as you change properties. Some of the property changes should affect multiple widgets.
  7. Click on the Save icon on the main toolbar to save the new file. The asterisks that indicates that the file is dirty should go away.
  8. Click on the X close button for the editor tab for this file
  9. Re-open the theme editor.
  10. Issue a Create HTML File command.
  11. Add some HTML widgets to the canvas along with TextBox to the canvas.
  12. Note that in 0.6, there is a browser caching problem. Theme editor changes might not be visible on the page editor or preview-in-browser because the browser might be using an earlier cached version of some of the CSS files. Refreshing the browser usually causes the latest theme CSS files to be used.

Expected results

  1. As you change properties in the theme editor, the canvas should update so that the widgets reflect the styling changes you made.
  2. After save and reopen, the saved theme should open and its contents should be the same as how it was when you clicked on the Save button.
  3. After you have created your new HTML file with the widgets, you should see that the widgets have the styling changes you made in the theme editor.

Subwidget theming

Steps

  1. You'll want to setup a virgin user environment by creating a whole new user login or resetting the existing user login (e.g., delete users/ folder, restart server). Clear the browser cache. Launch the daVinci server. Login if necessary.
  2. Open the Dojo/Claro theme editor from the Open menu.
  3. Click to select the Calendar widget.
  4. In the Outline palette, select one of the subwidget children of Calendar, such as SelectedDate. Change Styling properties such as font size and font color. Only the styling of the selected date in the Calendar widget should show the styling changes.
  5. Repeat the process for one or more other subwidgets of Calendar.
  6. Repeat the process for one or more other subwidgets of other container widgets (e.g., TabContainer or TitlePane).
  7. Click on the Save icon on the main toolbar to save the new file. The asterisks that indicates that the file is dirty should go away.
  8. Click on the X close button for the editor tab for this file
  9. Re-open the theme editor.
  10. Press the Add file icon on the main toolbar to create a new HTML file.
  11. Add a Calendar and other widgets you modified in the theme editor to the canvas.
  12. Note that in 0.6, there is a browser caching problem. Theme editor changes might not be visible on the page editor or preview-in-browser because the browser might be using an earlier cached version of some of the CSS files. Refreshing the browser usually causes the latest theme CSS files to be used.

Expected results

  1. As you change properties in the theme editor, the canvas should update so that the widgets reflect the styling changes you made.
  2. After save and reopen, the saved theme should open and its contents should be the same as how it was when you clicked on the Save button.
  3. After you have created your new HTML file with the widgets, you should see that the widgets have the styling changes you made in the theme editor.

States theming

Steps

  1. You'll want to setup a virgin user environment by creating a whole new user login or resetting the existing user login (e.g., delete users/ folder, restart server). Clear the browser cache. Launch the daVinci server. Login if necessary.
  2. Open the Dojo/Claro theme editor from the Open menu.
  3. Click to select the TextBox widget.
  4. Change to the Hover state. (Right now, this is done in the Outline palette.)
  5. In the Styling palette, change some properties to restyle the hover state of the widget. For example, change some combination of font-size, font-color, and background color. You should see the effect on the canvas as you change properties.
  6. Click to select the Button widget.
  7. Change to the Hover state. (Right now, this is done in the Outline palette.)
  8. In the Styling palette, change some properties to restyle the hover state of the widget. For example, change some combination of font-size, font-color, and background color. You should see the effect on the canvas as you change properties.
  9. Click on the Save icon on the main toolbar to save the new file. The asterisks that indicates that the file is dirty should go away.
  10. Click on the X close button for the editor tab for this file
  11. In Explorer window, re-open the theme editor.
  12. Press the Add file icon on the main toolbar to create a new HTML file.
  13. Add a Button and TextBox widget to the canvas.
  14. Click on the Preview in Browser button on the main toolbar. Move the mouse to hover over the Button and then hover over the TextBox.

Expected results

  1. As you change properties in the theme editor, the canvas should update so that the widgets reflect the styling changes you made.
  2. After save and reopen, the saved theme should open and its contents should be the same as how it was when you clicked on the Save button.
  3. After you have created your new HTML file with the widgets, and then launched the Preview in Browser, you should see that the widgets have the styling changes you made in the theme editor.
